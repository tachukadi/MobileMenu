# Changelog

All notable changes to the MobileMenu WordPress plugin will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-12-15

### Added
- Initial release of MobileMenu plugin
- Mobile and tablet responsive menu support
- Multiple menu selection capability
- Custom icon support (Dashicons, Font Awesome, SVG)
- 8 menu open/close animation styles
- 3 submenu animation styles
- Full color customization
- Gradient background support
- Configurable animation speed and easing
- Hamburger icon positioning (4 positions)
- Icon positioning relative to text (above, left, right)
- Close button option
- Auto-close on outside click
- Auto-close on anchor link click with smooth scroll
- Prevent body scroll when menu is open
- Blur background overlay option
- Submenu expand/collapse functionality
- ESC key to close menu
- Responsive breakpoint controls
- Touch-friendly interface
- Admin settings page with tabbed interface
- Icon picker modal with search
- SVG upload and paste support
- AJAX settings saving
- WordPress admin color picker integration
- Plugin activation/deactivation hooks
- Uninstall cleanup script
- Security: Nonce verification, capability checks, input sanitization
- Internationalization support (i18n ready)
- Custom JavaScript events for extensibility
- PHP 8.2+ compatibility
- OOP architecture
- WordPress Coding Standards compliance

### Features
- **General Settings**: Enable/disable menu, device targeting, breakpoint configuration
- **Menu Selection**: Choose from available WordPress menus
- **Icon Management**: Assign custom icons to individual menu items
- **Animations**: Rich selection of professional animations
- **Styling**: Complete color and typography control
- **Behavior**: Smart menu behavior options

### Technical
- Lightweight: ~50KB total assets
- No external dependencies (except jQuery from WordPress)
- Hardware-accelerated animations
- Mobile-first responsive design
- Cross-browser compatible
- WCAG accessibility considerations
